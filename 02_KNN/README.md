# K-Nearest Neighbors (KNN)

## Overview

This project demonstrates the implementation of the K-Nearest Neighbors (KNN) algorithm for classification.

## Concepts Learned

* Data Loading
* Data Preprocessing
* Train-Test Split
* Feature Scaling
* KNN Classification
* Model Evaluation
* Prediction

## Libraries Used

* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Workflow

1. Load Dataset
2. Explore Data
3. Preprocess Data
4. Split Data
5. Scale Features
6. Train KNN Model
7. Make Predictions
8. Evaluate Accuracy

## Learning Outcome

Learned how distance-based classification works using the K-Nearest Neighbors algorithm.
